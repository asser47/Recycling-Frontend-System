## üîß TROUBLESHOOTING

### Common Issues & Solutions

**Issue**: Classes not working  
**Solution**: Check `src/styles.css` for class definition, use Bootstrap equivalent

**Issue**: Responsive not working  
**Solution**: Ensure `.container` wraps content, use proper `.col-*` classes

**Issue**: Colors not matching  
**Solution**: Use CSS variables from `:root`, check color hex values

**Issue**: Spacing inconsistent  
**Solution**: Use standardized spacing classes (p-*, m-*, gap-*)

**Issue**: Buttons not styled  
**Solution**: Add `.btn` class first, then `.btn-primary` (or other variant)
